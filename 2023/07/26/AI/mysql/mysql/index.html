  • drop database if exists testdb3; 经常和创建表时结合使用
  1. 创建表

  1. 更改表

tips：

  • 主键：表中经常有一个列或列的组合，其值能唯一地标识表中的每一行

4.SQL 语句

tips：

  • select name as '姓名' from t_user ；将 name 替换为 姓名 显示
  • where 子句
    age >10
    is null
    and , or , between
  • 聚合函数
    max() min() avg() sum() count()
    floor() 向下取整
    ceil() 向上取整
  • 分组
    group by 后面只能和 having 联用，不能使用 where
    select 后同时含有聚合函数和字段时使用
  • 排序
    asc 正序
    desc 倒序
